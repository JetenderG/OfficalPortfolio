Lyrify

The very first site that I worked on with a group. This site allows users to search for song or artist and get certain information such as albums, sample songs, and lyrics.Relatively a simple project that allows to use bare minimum to create a good site. Most of the other scripts, css, and other such was done by other members. For most I have developed the Firebase portion of the site where I created a top song and artist list by users input.

Tech Specs:

  • spotify-api
  • Firebase
  • bootstrap
  • Javscript
  • Ajax

The one thing that you need is a spotify web

Word Guesser

First Independent project done through the simple use of html,css,boostrap, and javascript.
